# # 设置短剧变现类型

> 接口应在服务器端调用，不可在前端（小程序、网页、APP等）直接调用，具体可参考接口调用指南。

接口英文名：developerSetIaaDrama

在视频号推荐、视频号入口展示、播放器选集等场景，对IAA/IAP/IAAP有不同的精准策略。若不进行配置/配置出错，将影响你的营收和流量分配。单次请求限制最多设置50个短剧。

- 本接口支持第三方平台代商家调用。
- 该接口所属的权限集 id 为：157
- 服务商获得其中之一权限集授权后，可通过使用 authorizer_access_token 代商家进行调用，具体可查看 第三方调用 说明文档。
