# # 小程序品牌申请状态查询

> 接口应在服务器端调用，不可在前端（小程序、网页、APP等）直接调用，具体可参考接口调用指南。

接口英文名：getFamousBrandApplyStatus

本接口用于查询小程序品牌申请状态

## # 1. 调用方式

### # HTTPS 调用

### # 云调用

- 本接口不支持云调用。

### # 第三方调用

- 本接口支持第三方平台代商家调用。
- 该接口所属的权限集 id 为：142
- 服务商获得其中之一权限集授权后，可通过使用 authorizer_access_token 代商家进行调用，具体可查看 第三方调用 说明文档。

## # 2. 请求参数

### # 查询参数 Query String Parameters

| 参数名 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- |
| access_token | string | 是 | ACCESS_TOKEN | 接口调用凭证，可使用 access_token、authorizer_access_token |

### # 请求体 Request Payload

无

## # 3. 返回参数

### # 返回体 Response Payload

| 参数名 | 类型 | 示例 | 说明 |
| --- | --- | --- | --- |
| errcode | number | 0 | 错误码 |
| errmsg | string | ok | 错误原因 |
| progress | object | - | 品牌申请进度 |
| application | object | - | 申请品牌信息 |

品牌申请进度

| 参数名 | 类型 | 说明 | 枚举 |
| --- | --- | --- | --- |
| status | number | 品牌申请状态 | 枚举值 |

### # Res.application Object Payload

申请品牌信息

| 参数名 | 类型 | 示例 | 说明 | 枚举 |
| --- | --- | --- | --- | --- |
| apply_for | number | 1 | 申请类型 | 枚举值 |

| status | number | - | 品牌申请状态 | 枚举值 |

### # Res.application.audit_info Object Payload

申请品牌信息
