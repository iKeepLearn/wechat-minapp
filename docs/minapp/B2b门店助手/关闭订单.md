# # 关闭订单

> 接口应在服务器端调用，不可在前端（小程序、网页、APP等）直接调用，具体可参考接口调用指南。

接口英文名：closeB2bOrder

仅当订单处于待支付状态（ORDER_PRE_PAY）时，可以调用本接口主动关闭订单。对于合单支付的订单，任一子单关单会使同主单下的所有子单都被关闭。

### # 云调用

### # 第三方调用

- 本接口支持第三方平台代商家调用。
- 该接口所属的权限集 id 为：158
- 服务商获得其中之一权限集授权后，可通过使用 authorizer_access_token 代商家进行调用，具体可查看 第三方调用 说明文档。
