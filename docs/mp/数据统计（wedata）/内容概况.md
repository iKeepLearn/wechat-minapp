# # 获取发表内容概况总数据

[调试诊断](https://developers.weixin.qq.com/console/devtools/debug?utm_source=api_tools)

> 接口应在服务器端调用，不可在前端（小程序、网页、APP等）直接调用，具体可参考接口调用指南。

接口英文名：getbizsummary

本接口用于获取圈选日期内，该账号下发表内容汇总的概览数据。

- 统计方式为：[开始日期-结束日期] 期间内（包含开始日期和结束日期），每天发表内容的指标的概览数据。

### # 云调用

- 调用方法：datacube.getbizsummary

### # 第三方调用

- 本接口支持第三方平台代商家调用。
- 该接口所属的权限集 id 为：7
- 服务商获得其中之一权限集授权后，可通过使用 authorizer_access_token 代商家进行调用，具体可查看 第三方调用 说明文档。

## # 4. 注意事项

- 数据存储起始时间为2025-11-01，之前日期数据无效。
- 查询日期范围的长度最长支持查询30天。
- 数据可能存在延迟，请以接口返回数据为准。
- 建议在次日8点后请求前1天数据。
