# # 获取被动回复概要数据

[调试诊断](https://developers.weixin.qq.com/console/devtools/debug?utm_source=api_tools)

> 接口应在服务器端调用，不可在前端（小程序、网页、APP等）直接调用，具体可参考接口调用指南。

接口英文名：getinterfacesummary

本接口用于获取被动回复概要数据

## # 1. 调用方式

### # HTTPS 调用

### # 云调用

- 调用方法：officialAccount.datacube.getInterfaceSummary
- 出入参和 HTTPS 调用相同，调用方式可查看 云调用 说明文档。

### # 第三方调用

- 本接口支持第三方平台代商家调用。
- 该接口所属的权限集 id 为：3
- 服务商获得其中之一权限集授权后，可通过使用 authorizer_access_token 代商家进行调用，具体可查看 第三方调用 说明文档。

## # 2. 请求参数

### # 查询参数 Query String Parameters

| 参数名 | 类型 | 必填 | 说明 |
| --- | --- | --- | --- ||
| access_token | string | 是 | 接口调用凭证，可使用 access_token、authorizer_access_token |

### # 请求体 Request Payload

| 参数名 | 类型 | 必填 | 示例 | 说明 |
| --- | --- | --- | --- | --- ||
| begin_date | string | 是 | 2014-12-07 | 起始日期(YYYY-MM-DD格式) |
| end_date | string | 是 | 2014-12-07 | 结束日期(最大时间跨度30天) |

## # 3. 返回参数

### # 返回体 Response Payload

| 参数名 | 类型 | 说明 |
| --- | --- | --- ||
| list | object | 数据列表 |

### # Res.list Object Payload

数据列表

| 参数名 | 类型 | 说明 |
| --- | --- | --- ||
| ref_date | string | 数据的日期 |
| callback_count | number | 通过服务器配置地址获得消息后，被动回复用户消息的次数 |
| fail_count | number | 上述动作的失败次数 |
| total_time_cost | number | 总耗时，除以callback_count即为平均耗时 |
| max_time_cost | number | 最大耗时 |
